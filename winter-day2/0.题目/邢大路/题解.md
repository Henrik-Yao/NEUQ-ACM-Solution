# 题解
## 7.1
    根据完全二叉树以及先序遍历的性质，一个节点下标为x，下一个遍历点则是则左儿子2*x，一直到虚节点，开始遍历右儿子2*x+1，根据这个性质递归建立树即可。当一个节点值为0时，代表它没有儿子，就return。使用结构体同时记录节点的值和高度，然后遍历所有的点，按照层数加和比较即可。
