## **7-1 求解区间覆盖问题**

输入数据后先将各个区间从小到大排序，求出各个区间的间隔，将间隔最短的位置用线段连接作为最优解，取前n-m个最短间隔连接，结果再加上每个区间的长度1。即:

int ans(int n, int m)
{
    int s = 0;
    for (int i = 0; i < n - m; i++) s += b[i];//取前n-m个最短间隔连接
    s += n;
    return s;
}
