一个小优化是预先筛出来$2-10000$以内的质数，然后每次询问直接枚举，省去了重复$check$的时间。

如果你的埃筛过不去，检查一下是不是写成$O(nlogn)$的了

### 7-4：

快速幂模板题，没什么好说的，贴个板子吧。

```cpp
typedef long long ll;
ll Pow(ll a, ll b, ll p)
{
	ll ans = 1 % p;
	for(; b; b >>= 1)
	{
		if(b & 1) ans = ans * a % p;
		a = a * a % p;
	}
	return ans;
}
```

要注意模数可能为1
