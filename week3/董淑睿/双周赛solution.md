# 7-3 汉诺塔III 
步骤如下：
假设n层在左边
1. 将n-1层放到最右边
2. 将第n层放到中间
3. 将n-1层放到最左边
4. 将第n层放到中间
5. 将n-1层放到最右边
最后可以找到通项。
# 7-4 汉诺塔II
步骤如下
设F(n)为四个柱子的汉诺塔
设f(n)为三个柱子的汉诺塔
F(n)=f(n-x)+F(x)*2
先把x个盘子用四个主子移动到一个塔上，再把剩下的盘子用三个柱子移动到目标上，最后把x个盘子用四个柱子移回去。
