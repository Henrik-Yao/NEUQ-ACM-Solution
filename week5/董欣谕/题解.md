2. dfs模板+pre数组存储路径

   首先要理解题意：从每个未访问的点进行dfs直到没有点可访问，然后进行下一个点的dfs

   每个点的dfs遍历：

   	for (int i=0; i<n; i++)
   		if (!vis[i]) {
   			vis[i]=1;
   			pre[++cnt]=i;
   			dfs(i);
   		}

   在dfs的时候存储路径：

   		if (!vis[v]) {
   			vis[v]=1;
   			pre[++cnt]=v;
   			dfs(v);
   		}
